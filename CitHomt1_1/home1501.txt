
First level: 1. Как отказаться от изменений в файле (файлах) находящийся в рабочем каталоге (work directory)?

Git checkout . (удаляются все изменения в измененных файлах)  или git checkout и название файла.


2.	Как вернуть файл из индекса в рабочий каталог?

Git restore --staged название файла   
С такой командой просто возвращается файл из индекса. В него после возвращения необходимо вносить изменения вручную (если нет необходимости, например, отменить все изменения)
Если необходимо отменить все изменения, то используем далее git checkout и название файла


3.	Как отменить ошибочный commit ?
Если до команды push
Git commit –amend (файл можно изменить через редактор, если надо просто дополнить, например)
Или
Git reset HEAD~1 или git reset HEAD (номер коммита)

Если после push
Git revert HEAD (вызывается редактор по умолчанию, соответственно, можно редактировать или сразу выйти не меняя)

Перед возвратами желательно использовать git log, чтоб четко понимать какие конкретно коммиты будут сейчас возвращены.
